Online Cloud HTTP RESTful API:

URI

servers/

Method: GET

Action

Get all the Servers for the authenticated user.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Servers.

Method: POST

Action

Add (create) a new Server for the authenticated user.

Request Body

JSON describing the new Server.

Required Fields
  • name - The name of the new Server.
  • address - The IP address of the new Server. This field must be unique for the user.
  • port - The CoAP port used by the new Server.
  • multicast - A boolean stating if the new Server is using a multicast address for CoAP communication or not.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Servers of the authenticated user.
  • 400 BAD REQUEST - Something is wrong with the request.
