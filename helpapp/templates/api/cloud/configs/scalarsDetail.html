Online Cloud HTTP RESTful API:

URI

configs/scalars/scalar_value_type_id/

Method: GET

Action

Get the Scalar Value Type identified with the ID scalar_value_type_id.

Possible Response Codes and Body

  • 200 OK - JSON object with the Scalar Value Type identified with ID scalar_value_type_id.
  • 404 NOT FOUND - If the requested Scalar Value Type doesn’t exist for the authenticated user.

Method: DELETE

Action

Delete the Scalar Value Type identified with the ID scalar_value_type_id.

Possible Response Codes and Body

  • 204 NO CONTENT - Empty Response confirming the Scalar Value Type deletion.
  • 404 NOT FOUND - If the requested Scalar Value Type doesn’t exist for the authenticated user.
Note: Each user can only individually get/delete his/her own custom Scalar Value Types, for core ones an unauthorized error will be presented and for other user custom Scalar Value Types a 404 error will be presented.
