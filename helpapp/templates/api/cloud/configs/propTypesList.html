Online Cloud HTTP RESTful API:

URI

configs/property_types/

Method: GET

Action

Get all the Property Types (both core and custom) for the authenticated user.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Property Types.

Method: POST

Action

Add (create) a new Property Type.

Request Body

JSON describing the new Property Type.

Required Fields
  • name - The name of the new Device Type.
  • value_type_class - “SCALAR” | “ENUM”.
  • value_type_id - The ID of the Value Type to which the new Property Type will be referring. This ID must exist on the Value Types subset defined by the field value_type_class.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Property Types.
  • 400 BAD REQUEST - Something is wrong with the request.
