Online Cloud HTTP RESTful API:

URI

configs/device_types/device_type_id/

Method: GET

Action

Get the Device Type identified with the ID device_type_id.

Possible Response Codes and Body

  • 200 OK - JSON object with the Device Type identified with ID device_type_id.
  • 404 NOT FOUND - If the requested Device Type doesn’t exist for the authenticated user.

Method: DELETE

Action

Delete the Device Type identified with the ID device_type_id.

Possible Response Codes and Body

  • 204 NO CONTENT - Empty Response confirming the Device Type deletion.
  • 404 NOT FOUND - If the requested Device Type doesn’t exist for the authenticated user.
Note: Each user can only individually get/delete his/her own custom Device Types, for core ones an unauthorized error will be presented and for other user custom Device Types a 404 error will be presented.
