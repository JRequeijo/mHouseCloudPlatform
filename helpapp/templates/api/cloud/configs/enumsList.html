Online Cloud HTTP RESTful API:

URI

configs/enums/

Method: GET

Action

Get all the Enumerated Value Types (both core and custom) for the authenticated user.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Enumerated Value Types.

Method: POST

Action

Add (create) a new Enumerated Value Type.

Request Body

JSON describing the new Enumerated Value Type.

Required Fields
  • name - The name of the new Enumerated Value Type.
  • default_value - A Choice ID that indicates the value by default for the new Enumerated Value Type. This value (ID) must be present on the choices field list.
  • choices - A non-empty list of Choice IDs that identify the Enumerated Value Type possible choices.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Enumerated Value Types.
  • 400 BAD REQUEST - Something is wrong with the request.
