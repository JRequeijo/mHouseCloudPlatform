Online Cloud HTTP RESTful API:

URI

configs/choices/

Method: GET

Action

Get all the Choices (both core and custom) for the authenticated user.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Choices.

Method: POST

Action

Add (create) a new Choice.

Request Body

JSON describing the new Choice.

Required Fields
  • name - The name of the new Choice.
  • value - A value that can be intrinsically related to the physical action/thing.

Possible Response Codes and Body

  • 200 OK - JSON object with all the Choices.
  • 400 BAD REQUEST - Something is wrong with the request.
