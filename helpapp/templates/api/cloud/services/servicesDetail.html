Online Cloud HTTP RESTful API:

URI

services/service_id/

Method: GET

Action

Get the Service identified with the ID service_id.

Possible Response Codes and Body

  • 200 OK - JSON object with the Service identified with ID service_id.
  • 404 NOT FOUND - If the requested Service doesn’t exist for the authenticated user.

Method: PUT/PATCH

Action

Update the Service identified with the ID service_id.

Request Body

JSON with the Service updated fields.

Required Fields (for PUT method only)
See request body for POST services/

Possible Response Codes and Body

  • 200 OK - JSON object with the Service identified with ID service_id.
  • 404 NOT FOUND - If the requested Service doesn’t exist for the authenticated user.

Method: DELETE

Action

Delete the Service identified with the ID service_id.

Possible Response Codes and Body

  • 204 NO CONTENT - Empty Response confirming the Service deletion.
  • 404 NOT FOUND - If the requested Service doesn’t exist for the authenticated user.
